# font-server

> 一个基于vue+koa+mongoose的简易博客

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

##功能
1. 文章列表
2. 文章查看

##计划
1. 显示文章
2. 文章评论、点赞、浏览
3. 留言
